app-samba
===========

## 运行在魔豆路由器上的基于samba协议的文件共享应用

    * 将移动硬盘（u盘）中的内容以加密／开放的方式共享到局域网中

## 功能

    * 支持安卓设备的正常访问。
    * 支持应用市场安装和升级；
    * 支持屏幕上开启／关闭共享服务；
    * 支持多分区硬盘的多模式共享，即可将特定分区设置为“开放共享”或者“加密共享”；
    * 支持屏幕上随意修改加密共享的密码;
