modou-samba
===========
基于samba协议的文件共享

最新版本：0.2
------------
>  更新记录

* 支持应用市场安装和升级；
* 支持屏幕上开启／关闭共享服务；
* 支持多分区硬盘的多模式共享，即可将特定分区设置为“开放共享”或者“加密共享”；
* 支持屏幕上随意修改加密共享的密码;

>  已知缺陷

* 服务开启状态下修改共享方式，需点“关闭服务”，再点“开启服务”；
* 在屏幕上开关服务，不能同步到web下的应用管理，导致状态会不一致.

安装/卸载
--------
* 进入魔豆路由web管理后台的“应用”界面,上传/卸载，或者应用市场中安装；
